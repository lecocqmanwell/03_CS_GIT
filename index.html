How to use Git...

Local Changes

git status

Changed in working directory

git diff

Tracked file changes

git add file1 file2 file3

Add changed files

git rm file
git rm dir/ -r
(recursive under directory)

Remove files

git diff --cached

See files ready for commit

git commit
git commit -m "My message"
git commit -a -m "My Message"
(tracked files only, auto add)

Commit changes

git commit --amend

Change last commit

git checkout -- file

Revert changes to file

Remote Update / Publish

git remote -v

List Remote

git remote show remote

Show information

git remote add path/url

Add remote

git fetch remote

Fetch changes

git push remote :branch

Publish tags

Branching/Tagging

git branch

List branches

git checkout branch

Switch to branch

git branch new

Create new branch

git branch new existing

Create branch from existing

git branch -d branch

Delete branch

git tag tagname

Tag current commit

History

git log

Show all commits

git log --pretty=-short

Short Format

git log -p

Patches

git log file

Show file commits

git log dir/

Show directory commits

git log --stat

Stats

git blame file

Who changed file
